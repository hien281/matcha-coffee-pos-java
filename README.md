# 🍵 Matcha Coffee POS System

Một ứng dụng Quản lý Bán hàng (Point of Sale) dành cho quán cà phê, được phát triển trên nền tảng ứng dụng Desktop.

## 🚀 Chức năng nổi bật
* Quản lý thực đơn và thao tác đặt món (Order) nhanh chóng.
* Tính tiền, tính tiền thừa và xuất hóa đơn trực tiếp.
* Quản lý nhân sự và ca làm việc (Shift Log).

## 🛠 Công nghệ & Công cụ sử dụng
* **Ngôn ngữ lập trình:** Java
* **Giao diện (GUI):** Java Swing

## 📥 Hướng dẫn cài đặt & Khởi chạy
1. Tải file cài đặt `.jar` phiên bản mới nhất tại phần **[Releases](https://github.com/hien281/matcha-coffee-pos-java/releases/tag/v1.0)**.
2. Ứng dụng yêu cầu máy tính đã cài đặt sẵn môi trường **Java (JDK/JRE)**.
3. Mở Command Prompt (CMD) hoặc Terminal và chạy dòng lệnh sau:

```bash
java -jar "LTUD.MATCHA.COFFEE.POS.APP.jar"
